## 🚀 Ce que je construis

Développeur Discord & full-stack, j'aime concevoir des bots qui rendent les serveurs vivants — de la modération à l'IA, en passant par le web. Je gère tout : le bot, le dashboard, le site, et le VPS derrière.

- 🔭 Mon projet phare : **[Kalvyx](https://kalvyx.com)** — un bot Discord tout-en-un **propulsé par Claude (Anthropic)** : modération, tickets, niveaux, IA, dashboard web.
- 🌱 J'apprends en construisant, du frontend jusqu'à l'ops serveur.
- 💬 Trouve-moi sur [Discord](https://discord.com/users/689890476811354242) ou [X](https://x.com/matgordfr).
